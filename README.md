A simple Employee Management System built using Python that allows users to perform CRUD operations and manage employee records with file storage support (JSON&amp; CSV).
# 🧑‍💼 Employee Management System (EMS)

## 📌 Overview
The Employee Management System (EMS) is a command-line based Python application designed to manage employee records efficiently. This project demonstrates fundamental programming concepts such as data structures, functions, control flow, and file handling.

The system allows users to perform all basic operations required to manage employee data, including adding, viewing, searching, updating, and deleting records.
## 🎯 Objectives
- To implement a menu-driven application using Python  
- To understand dictionary-based data storage  
- To perform CRUD (Create, Read, Update, Delete) operations  
- To implement file handling using JSON and CSV formats  
## 🚀 Features
- ➕ Add new employee  
- 📋 View all employees  
- 🔍 Search employee by ID  
- ✏️ Update employee details  
- 🗑️ Delete employee  
- 💾 Save data to JSON file  
- 📊 Save data to CSV file  
## 🛠️ Technologies Used
- Python  
- Built-in Data Structures (Dictionary)  
- File Handling (JSON, CSV)  
## 🧠 Concepts Covered
- Functions (Modular Programming)  
- Loops (`while`, `for`)  
- Conditional Statements (`if-else`)  
- Dictionary operations  
- Input/Output handling  
- File operations (read/write)
